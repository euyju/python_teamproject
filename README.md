# 프로젝트 개요

- 음식 사진을 입력하면(test_imeages)  
- AI가 음식 종류를 예측하고(predict.py)  
- 해당 음식의 영양정보(FoodAPI.py)까지 출력(main)해주는 시스템입니다.
